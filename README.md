# powerbi-dashboard-vendas
Dashboard de análise de vendas desenvolvido em Power BI com foco em indicadores de desempenho.

📊 Dashboard de Vendas – Power BI

📌 Objetivo:
Desenvolver um dashboard interativo para análise de desempenho comercial, permitindo acompanhar faturamento, volume de vendas e performance por região.

🛠 Ferramentas Utilizadas:
Power BI
Modelagem de Dados
DAX
Excel (tratamento de base)
Color Hex (design)
ColorSpace (design)

📈 Indicadores Analisados:
Faturamento total
Quantidade vendida
Produto mais vendido
Ticket médio
Performance por marca
Análise geográfica por país
Evolução mensal de vendas

📊 Análises Realizadas:
O dashboard permite identificar padrões de crescimento, sazonalidade e regiões com maior potencial de expansão, além de facilitar o acompanhamento de metas e desempenho comercial.

🎯 Conclusão:
A construção do dashboard teve como foco transformar dados brutos em informações visuais e estratégicas para apoiar a tomada de decisão e o acompanhamento intuitivo da performance geral do negócio.

📂 Arquivo do Projeto:
O arquivo .pbix está disponível neste repositório para visualização completa do modelo de dados e medidas DAX.

📊 Base de Dados:
A base utilizada no projeto é fictícia e está disponível na pasta _data_ deste repositório.
